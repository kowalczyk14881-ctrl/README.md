# Мой смарт-контракт Tether (USDT)

<img src="https://raw.githubusercontent.com/kowalczyk14881-ctrl/assets/master/logo.png" width="120"/>


Это смарт-контракт для работы с токеном Tether (USDT).

## Описание контракта

- Прием платежей в USDT
- Отправка токенов на другой адрес
- Проверка баланса

## Использование

1. Подключите контракт к сети Ethereum/Polygon/BSC.
2. Вызовите функции `transfer`, `balanceOf` и т.д.
3. Следите за балансом через ваш кошелек.
